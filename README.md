## Development
### Frontend
install dependencies:

```bash
npm install
```

start it with:

```bash
npm run dev
```

front end will be running in <http://localhost:3000>

### Backend
install the dependencies:

```bash
pip install -r requirements.txt
```
start it with:

```bash
flask --app server run
```
